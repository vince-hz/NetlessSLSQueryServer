# How to start

Create `env.json` on the root directory.

```json
{
    "ProjectName": "xxxxxx",
    "Endpoint": "xxxxxx",
    "LogStoreName": "xxxxxx",
    "AccessKeyID": "xxxxxx",
    "AccessKeySecret": "xxxxxx",
    "WhiteOperationToken": "xxxxxx"
}
```

Then, run `go run main.go` on the root directory.
